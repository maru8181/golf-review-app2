## README

### Golfer

ゴルフ場のレビューが行えるサイトです。  
URL(https://）  
トップ画面からゲストユーザーでログインが可能です。

### 主な機能,使用技術,gemなど

- ユーザー登録(device)
- ユーザー間でのフォロー、フォロワー機能(ajax)
- レビュー機能(CRUD)
- 画像投稿機能(carrierwave)
- いいね機能(ajax)
- 検索機能(ransack)

### 使用技術

- Ruby 2.6.3
- Rails 6.0.2
- MYSQL 8.0
- GitHub,Git
- Docker(開発環境)
- AWS()
